# سعد - محفظة أعمال شخصية

هذا المشروع هو موقع ويب شخصي تفاعلي ومتطور لسعد الدوسري، تم بناؤه باستخدام React و TypeScript و Tailwind CSS.

## للتشغيل المحلي (Local Development)

1. **تثبيت المكتبات:**
   تأكد من أن لديك Node.js و npm مثبتين. ثم قم بتشغيل الأمر التالي لتثبيت جميع الاعتماديات:
   \`\`\`bash
   npm install
   \`\`\`

2. **إعداد مفتاح API للمساعد الذكي:**
   - أنشئ ملف \`.env\` في جذر المشروع.
   - احصل على مفتاح API من [Google AI Studio](https://aistudio.google.com/app/apikey).
   - أضف المفتاح إلى ملف \`.env\` كالتالي:
     \`\`\`
     VITE_GEMINI_API_KEY=your_actual_api_key_here
     \`\`\`

3. **تشغيل خادم التطوير:**
   \`\`\`bash
   npm run dev
   \`\`\`
   سيتم فتح الموقع في متصفحك على العنوان \`http://localhost:3000\`.

## البناء للنشر (Build for Production)

لإنشاء نسخة جاهزة للنشر على استضافة، قم بتشغيل الأمر:
\`\`\`bash
npm run build
\`\`\`
سيتم إنشاء مجلد \`dist\` يحتوي على جميع ملفات الموقع الجاهزة.
